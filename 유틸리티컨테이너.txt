유틸리티 컨테이너란?

특정 환경만 포함하는 컨테이너를 의미한다.

사용하는 이유
 
 호스트 머신에 툴을 전역적으로 설치하지 않고 애플리케이션에 필요한 환경이 포함된 컨테이너를 사용
 
 
 docker exec : 이 컨테이너가 실행하는 기본 명령 외에 실행 중인 컨테이너 내에 특정 명령을 실행할 수 있다. 따라서 Dockerfile에 지정되어 있는 명령 외에 입력된 특정 명령을 실행하게 된다. 컨테이너 내부에 작성된 로그파일을 읽는데 유용

docker exec  -it 컨테이너이름 npm init 

docker run -it node npm init
